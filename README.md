## Nome del Progetto / Project Name

Stabilizzazione del pendolo inverso / Stabilization-Reverse-Pendulum

---

### Scopo del Progetto / Project Aim

(IT) : Linearizzare il sistema e stabilizzarlo attorno al punto d'equilibrio instabile

(EN) : To linearize the system and to stabilize it around unstable equilibrium point

---

### Linguaggi e tecnologie / Languages & Technologies

- Matlab + Simulink

---

### Utilizzo / Usage

(IT) : Apri in matlab il file livescript .mlx d'interesse e premi su run. Usa i file Simulink per le simulazioni.

(EN) : Open in matlab the desidered livescript file .mlx and then click on 'run'. Use Simulink files for simulations.
